

# DNSレコード


> [!NOTE] DNSレコードのTTLとは
> キャッシュサーバがキャッシュを保存する期限（単位：秒）


## レコードタイプ

|       |                                           |
| ----- | ----------------------------------------- |
| A     | IPv4アドレス Addre                            |
| AAAA  | IPv6アドレス                                  |
| CNAME | ドメイン名やホスト名の別名を定義する **C**anonical **NAME** |
| PTR   | IPアドレスに対応するドメイン名を定義する<br>逆引きに利用される        |
| SOA   |                                           |
| SPF   |                                           |
| TXT   |                                           |
