

# DNSレコード


> [!NOTE] DNSレコードのTTLとは
> キャッシュサーバがキャッシュを保存する期限（単位：秒）


## レコードタイプ

|       |                                    |
| ----- | ---------------------------------- |
| A     | IPv4アドレス                           |
| AAAA  | IPv6アドレス                           |
| CNAME |                                    |
| PTR   | IPアドレスに対応するドメイン名を定義する<br>逆引きに利用される |
| SOA   |                                    |
| SPF   |                                    |
| TXT   |                                    |
